# Overview

As a growing software engineer, I am continually working to strengthen my understanding of different programming languages and development concepts. This project allowed me to explore Kotlin a modern, concise language widely used for Android development and backend applications. My goal was to build a functioning console application that demonstrates core Kotlin features and improves my problem-solving skills.

For this project, I created a simple console-based Task Manager. The software allows users to add, list, and remove tasks using a clean and interactive command-line interface. It uses Kotlin classes, mutable lists, input handling, loops, and control structures to manage basic task operations.

My purpose for writing this software was to practice Kotlin syntax, understand how Kotlin handles collections, and get comfortable building interactive console applications. This project helps lay the foundation for more advanced Kotlin development in the future.
